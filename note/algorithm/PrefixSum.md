### 一、设计技巧
**1. 为 $a[n]$ 创建差分数组 $diff[n + 1]$**
多一个元素 **$diff[n]$** 可避免区间 **$[L, R]$** ​加值时的边界判断（当 **$R = n - 1$** 时访问 **$diff$** 数组会越界）
- **$diff[0] = a[0], \; diff[i] = a[i] - a[i-1] \; (0 \lt i \lt n)$**。也可以全部初始化为0，只记录新增的变化
- 对子数组 **$[L, R] \; (0 \le L \le R \lt n)$** 所有元素加上 **$val$** ​时有 **$$diff[L] \gets diff[L] + val, \; diff[R+1] \gets diff[R+1] - val$$**
- **$a[k] = \sum_{i=0}^{k}diff[i] \; (0 \le k \lt n)$**

**2. 为 $a[n]$ 创建前缀和数组**
- 创建 **$sum[n + 1]$**，添加代表前 0 个元素的前缀和 **$sum[0] = 0$**，此时 **$sum[i]$** 代表前 **$i$** 个元素的前缀和
  - **$sum[0] = 0, \; sum[i] = sum[i-1] + a[i - 1] \; (0 \lt i \le n)$**
- 创建 **$sum[n]$**，不添加代表前 0 个元素的前缀和，此时 **$sum[i]$** 代表前 **$i + 1$** 个元素的前缀和
  - **$sum[0] = a[0], \; sum[i] = sum[i-1] + a[i] \; (0 \lt i \lt n)$**
